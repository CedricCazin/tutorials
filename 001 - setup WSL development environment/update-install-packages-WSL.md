
# Prerequisites

Enter the WSL distribution

# Update packages

Update the distribution packages

```sh
sudo apt update && sudo apt upgrade
```

# Install packages

```sh
sudo apt-get install -y \
    git-all \
    bzip2 \
    jq \  
    iptables \
    unzip \
    wget \
    zlib1g \
    libssl-dev \
    zlib1g-dev \
    libnghttp2-dev \
    curl
```
